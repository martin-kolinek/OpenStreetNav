Detailed Description

template<typename C, typename B>
class xsd::cxx::tree::language< C, B >

Class corresponding to the XML Schema language built-in type.

The language class publicly inherits from and has the same set of constructors as std::basic_string. It therefore can be used as std::string (or std::wstring if you are using wchar_t as the character type).
