Detailed Description

template<typename C, typename B>
class xsd::cxx::tree::ncname< C, B >

Class corresponding to the XML Schema NCame built-in type.

The ncname class publicly inherits from and has the same set of constructors as std::basic_string. It therefore can be used as std::string (or std::wstring if you are using wchar_t as the character type).
